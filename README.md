# 🛡️ Digital Arrest Scam Shield

**ET AI Hackathon 2026 | Theme: Digital Public Safety**[cite: 1]

This repository houses an AI-powered intelligence platform designed to neutralize "Digital Arrest" hostage scams[cite: 1]. These industrialized operations use spoofed numbers and fake government portals to defraud citizens[cite: 1]. Our solution shifts the paradigm from reactive investigation to predictive threat neutralization by intervening *before* a financial transfer occurs[cite: 1].

### 🚀 The Solution
Built on a robust, high-throughput data engineering pipeline, the system ingests live audio and communication metadata to trigger autonomous interventions. 

* **Real-Time Data Streaming:** Processes live call chunks and VoIP routing metadata without latency.
* **Multi-Modal AI Fusion:** Combines speech-to-text (STT), voice spoofing detection, and NLP-based scam script classification[cite: 1].
* **Agentic Orchestrator:** Calculates a live "Fraud Probability Score" to instantly severe the connection if the threshold is breached[cite: 1]. 
* **Dual-View Dashboards:** Features a protective mobile overlay for the citizen and a real-time Command Centre dashboard for law enforcement[cite: 1].
